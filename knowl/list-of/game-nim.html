Game 1.6. Nim.

Nim is two-player heap game. Players take turns selecting a heap and removing any number of tokens from it. The player who takes the last token from the last remaining heap is the winner.
A common starting arrangement is to have one heap of size three, one heap of size five, and one heap of size seven. However, the game can start with any number of heaps with any number of tokens in them.
